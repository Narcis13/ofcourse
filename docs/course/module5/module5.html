üí° Event Bubbling Explained

Events "bubble up" through the DOM tree. If you click a button inside a div, both the button AND the div receive the click event!

To stop this, use event.stopPropagation()

To prevent default behavior (like form submission), use event.preventDefault()

üîÅ Loops - Repeat Actions!

Loops let you repeat code without writing it multiple times:

// For loop - when you know how many times
for (let i = 1; i <= 5; i++) {
    console.log("Lap " + i);
}

// While loop - repeat while condition is true
let energy = 100;
while (energy > 0) {
    console.log("Energy: " + energy);
    energy -= 20;
}

// For...of loop - go through arrays
const fruits = ["apple", "banana", "orange"];
for (const fruit of fruits) {
    console.log("I like " + fruit);
}
