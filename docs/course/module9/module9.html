‚öõÔ∏è React Basics: JSX, the Magic Syntax

React uses something called JSX - it looks like HTML but it's actually JavaScript! This is where the magic happens. ü™Ñ

// JSX lets you write HTML-like code in JavaScript!
const element = (
    <div className="greeting">
        <h1>Hello, {userName}!</h1>
        <p>Welcome to React</p>
    </div>
);

// It's actually JavaScript underneath:
const element = React.createElement(
    'div',
    { className: 'greeting' },
    React.createElement('h1', null, 'Hello, ', userName, '!'),
    React.createElement('p', null, 'Welcome to React')
);

üí° Props vs Regular Function Arguments

Props work just like function parameters, but with superpowers!

// Regular function
function greet(name, age) {
    return `Hello ${name}, you are ${age} years old`;
}

// React component with props
function Greeting({ name, age }) {
    return <h1>Hello {name}, you are {age} years old</h1>;
}
