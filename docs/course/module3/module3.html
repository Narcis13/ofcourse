üéØ CSS Selectors - Targeting Elements Like a Sniper

Before you can style something, you need to select it. CSS selectors are like coordinates that tell CSS exactly which HTML elements to style. Master these, and you can style ANYTHING!

/* Element Selector - Targets all elements of a type */
p {
    color: blue;
}

/* Class Selector - Targets elements with specific class */
.highlight {
    background-color: yellow;
}

/* ID Selector - Targets one specific element */
#header {
    font-size: 24px;
}

/* Descendant Selector - Targets nested elements */
nav a {
    text-decoration: none;
}

/* Multiple Classes */
.btn.primary {
    background: blue;
}

üí° CSS Specificity Rules

When multiple styles target the same element, specificity determines which wins:

  1. Inline styles (style="...") - 1000 points
  2. IDs (#header) - 100 points
  3. Classes (.highlight) - 10 points
  4. Elements (p, div) - 1 point

Higher specificity wins! If tied, the last rule wins.

üì¶ The Box Model - The Secret to CSS Spacing

Here's a mind-blowing fact: EVERYTHING in CSS is a box! Understanding the box model is like unlocking a superpower - suddenly, spacing and layout make perfect sense.

Margin
Border
Padding
Content
/* The Box Model Properties */
.box {
    /* Content width and height */
    width: 200px;
    height: 100px;
    
    /* Padding - Space inside the border */
    padding: 20px;  /* All sides */
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 10px;
    padding-left: 15px;
    /* Shorthand: top right bottom left */
    padding: 10px 15px 10px 15px;
    
    /* Border - The edge of the box */
    border: 2px solid black;
    border-radius: 8px;  /* Rounded corners! */
    
    /* Margin - Space outside the border */
    margin: 20px;
    margin: 10px 20px;  /* vertical horizontal */
    
    /* Pro tip: Include padding/border in width */
    box-sizing: border-box;
}

üìê Flexbox - Make Things Line Up Without Tears!

Remember trying to center things in CSS and wanting to cry? Flexbox is here to save the day! It's like having a magic wand that makes elements line up exactly how you want them.

/* Flexbox Container Properties */
.flex-container {
    display: flex;  /* Turn on flexbox! */
    
    /* Main axis direction */
    flex-direction: row;        /* Default: left to right */
    flex-direction: column;     /* Top to bottom */
    flex-direction: row-reverse;
    flex-direction: column-reverse;
    
    /* Alignment on main axis */
    justify-content: flex-start;  /* Default */
    justify-content: center;      /* Center items */
    justify-content: space-between;
    justify-content: space-around;
    justify-content: space-evenly;
    
    /* Alignment on cross axis */
    align-items: stretch;     /* Default */
    align-items: center;      /* Center vertically */
    align-items: flex-start;
    align-items: flex-end;
    
    /* Wrapping */
    flex-wrap: nowrap;    /* Default */
    flex-wrap: wrap;      /* Allow wrapping */
    
    /* Gap between items */
    gap: 20px;
}

/* Flex Item Properties */
.flex-item {
    flex: 1;           /* Grow to fill space */
    flex-grow: 2;      /* Grow twice as much */
    flex-shrink: 0;    /* Don't shrink */
    flex-basis: 200px; /* Starting size */
}

üî≤ CSS Grid - Two-Dimensional Layouts Made Easy

While Flexbox is great for one-dimensional layouts (rows OR columns), CSS Grid is perfect for two-dimensional layouts (rows AND columns). It's like having a spreadsheet for your layout!

/* Grid Container */
.grid-container {
    display: grid;
    
    /* Define columns */
    grid-template-columns: 200px 200px 200px;  /* 3 fixed columns */
    grid-template-columns: 1fr 1fr 1fr;        /* 3 equal columns */
    grid-template-columns: 1fr 2fr 1fr;        /* Middle column twice as wide */
    grid-template-columns: repeat(3, 1fr);     /* Shorthand for equal columns */
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Responsive! */
    
    /* Define rows */
    grid-template-rows: 100px 200px;
    grid-template-rows: auto auto;  /* Size to content */
    
    /* Gaps */
    gap: 20px;              /* Both row and column gaps */
    row-gap: 10px;
    column-gap: 20px;
}

/* Grid Items */
.grid-item {
    /* Span multiple columns/rows */
    grid-column: span 2;    /* Take up 2 columns */
    grid-row: span 2;       /* Take up 2 rows */
    
    /* Specific placement */
    grid-column: 1 / 3;     /* Start at 1, end at 3 */
    grid-row: 2 / 4;
}

üì± Responsive Design - One Site, All Devices

Your website needs to look amazing whether someone's viewing it on a phone, tablet, or desktop. Responsive design makes this happen with CSS media queries!

/* Mobile First Approach */
/* Base styles (mobile) */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet styles */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
        padding: 20px;
    }
}

/* Desktop styles */
@media (min-width: 1024px) {
    .container {
        width: 1000px;
        padding: 30px;
    }
}

/* Common Breakpoints */
/* Mobile: 0-767px (default) */
/* Tablet: 768px-1023px */
/* Desktop: 1024px+ */

/* Responsive Typography */
html {
    font-size: 16px;
}

@media (min-width: 768px) {
    html {
        font-size: 18px;
    }
}

/* Responsive Images */
img {
    max-width: 100%;
    height: auto;
}

/* Responsive Grid */
.grid {
    display: grid;
    grid-template-columns: 1fr;  /* Mobile: 1 column */
}

@media (min-width: 768px) {
    .grid {
        grid-template-columns: repeat(2, 1fr);  /* Tablet: 2 columns */
    }
}

@media (min-width: 1024px) {
    .grid {
        grid-template-columns: repeat(3, 1fr);  /* Desktop: 3 columns */
    }
}

‚ú® Transitions & Effects - Adding Life to Your Site

Static websites are so 2010! Let's add smooth transitions, hover effects, and animations that make your site feel alive and interactive.

/* Transitions - Smooth changes between states */
.button {
    background-color: #6366f1;
    color: white;
    padding: 10px 20px;
    border-radius: 4px;
    
    /* Transition syntax: property duration timing-function delay */
    transition: all 0.3s ease;
    transition: background-color 0.3s ease;
    transition: transform 0.2s ease-in-out;
    
    /* Multiple properties */
    transition: background-color 0.3s ease, transform 0.2s ease;
}

.button:hover {
    background-color: #4f46e5;
    transform: translateY(-2px);
}

/* Transform - Change position, size, rotation */
.element {
    transform: translateX(20px);     /* Move horizontally */
    transform: translateY(-10px);    /* Move vertically */
    transform: scale(1.2);           /* Make bigger */
    transform: rotate(45deg);        /* Rotate */
    transform: skew(10deg);          /* Skew */
    
    /* Combine transforms */
    transform: translateY(-5px) scale(1.05);
}

/* Animations - Complex, repeatable effects */
@keyframes bounce {
    0% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
    100% { transform: translateY(0); }
}

.bouncing {
    animation: bounce 2s ease-in-out infinite;
    /* name duration timing-function iteration-count */
}

/* Hover Effects */
.card {
    transition: all 0.3s ease;
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 10px 30px rgba(0,0,0,0.2);
}

/* Cool gradient animation */
@keyframes gradient {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}

.gradient-bg {
    background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
    background-size: 400% 400%;
    animation: gradient 15s ease infinite;
}
